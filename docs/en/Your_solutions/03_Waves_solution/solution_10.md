## 10. Wave Interference

**Problem:**
Two identical wave sources $S_1$ and $S_2$ oscillate in phase, producing waves with wavelength $\lambda$. 
1. What is the condition for **constructive interference** at a point $P$?
2. What is the condition for **destructive interference** at a point $P$?

**Solution:**

The interference at any point $P$ is determined by the **path difference** ($\Delta L$), which is the difference in distance the two waves travel to reach that point:
$$\Delta L = |L_2 - L_1|$$

### 10.1. Constructive Interference
Constructive interference occurs when the peaks (crests) of both waves align perfectly. This happens when the path difference is an integer multiple of the wavelength:
$$\Delta L = n\lambda \quad \text{where } n = 0, 1, 2, \dots$$
In this case, the waves reinforce each other, and the resulting amplitude is the sum of the individual amplitudes ($A_{total} = A_1 + A_2$).

### 10.2. Destructive Interference
Destructive interference occurs when the peak of one wave aligns with the trough of the other. This happens when the path difference is a half-integer multiple of the wavelength:
$$\Delta L = (n + \frac{1}{2})\lambda \quad \text{where } n = 0, 1, 2, \dots$$
In this case, the waves cancel each other out. If the amplitudes are equal, the resulting displacement is zero ($A_{total} = 0$).

**Answer:** * Constructive: $\Delta L = 0, \lambda, 2\lambda, \dots$
* Destructive: $\Delta L = \frac{\lambda}{2}, \frac{3\lambda}{2}, \frac{5\lambda}{2}, \dots$

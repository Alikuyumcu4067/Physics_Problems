## 11. Interference - Path Difference Analysis

**Problem:**
Two identical wave sources $S_1$ and $S_2$ oscillate in phase, producing waves with a wavelength $\lambda$. If the path difference $L_2 - L_1$ to a point $P$ is **$1.5\lambda$**, what kind of interference occurs at that point?

**Solution:**

The type of interference is determined by comparing the path difference ($\Delta L$) to the wavelength ($\lambda$).

**Step 1: Identify the given path difference**
$$\Delta L = 1.5\lambda = \frac{3}{2}\lambda$$

**Step 2: Compare with interference conditions**
* **Constructive Interference:** $\Delta L = n\lambda$ (where $n = 0, 1, 2, \dots$)
* **Destructive Interference:** $\Delta L = (n + \frac{1}{2})\lambda$ (where $n = 0, 1, 2, \dots$)

**Step 3: Test for $n$**
Let's check the destructive interference condition:
$$(n + \frac{1}{2})\lambda = 1.5\lambda$$
$$n + 0.5 = 1.5$$
$$n = 1$$

Since $n$ is an integer ($n=1$), the condition for destructive interference is perfectly met.

**Answer:**
At a path difference of **$1.5\lambda$**, the waves arrive exactly out of phase (crest meets trough). This results in **total destructive interference**, and the net displacement at point $P$ will be zero.

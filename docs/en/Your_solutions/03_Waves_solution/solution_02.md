## 2. String Harmonics – Formal Step-by-Step Solution

### **Problem Statement**
A guitar string is $64 \text{ cm}$ long and has a fundamental frequency (one antinode) of $330 \text{ Hz}$. What is the speed of the wave on this string?

---

### **Step 1: Identify and Convert Given Data**
To perform calculations using SI units (meters, seconds, Hertz), we must first convert the physical dimensions:
* **String Length ($L$):** $64 \text{ cm} = \mathbf{0.64 \text{ m}}$
* **Fundamental Frequency ($f_1$):** $\mathbf{330 \text{ Hz}}$
* **Harmonic Mode ($n$):** $n=1$ (The problem specifies "fundamental frequency" and "one antinode").

### **Step 2: Determine the Wavelength ($\lambda$) from the Boundary Conditions**
For a string of length $L$ fixed at both ends, a standing wave forms. The relationship between the string length and the wavelength for any harmonic $n$ is:
$$L = n \frac{\lambda_n}{2}$$

For the **fundamental frequency** ($n=1$), the string length is exactly half of one wavelength:
1.  **Formula:** $L = \frac{\lambda}{2}$
2.  **Rearranging for $\lambda$:** $\lambda = 2L$
3.  **Substitution:** $\lambda = 2 \times 0.64 \text{ m}$
4.  **Result:** $\lambda = \mathbf{1.28 \text{ m}}$

### **Step 3: Calculate the Wave Speed ($v$)**
Now that we have the wavelength ($\lambda$) and the frequency ($f$), we apply the universal wave equation:
1.  **Formula:** $v = f \cdot \lambda$
2.  **Substitution:** $v = 330 \text{ Hz} \times 1.28 \text{ m}$
3.  **Calculation:** $v = 422.4 \text{ m/s}$

---

### **Step 4: Summary Table**
| Parameter | Symbol | Value |
| :--- | :--- | :--- |
| Length | $L$ | $0.64 \text{ m}$ |
| Harmonic Number | $n$ | $1$ |
| Wavelength | $\lambda$ | $1.28 \text{ m}$ |
| Frequency | $f$ | $330 \text{ Hz}$ |
| **Wave Speed** | **$v$** | **$422.4 \text{ m/s}$** |

---

### **Step 5: Physical Analysis**
The calculated speed ($422.4 \text{ m/s}$) represents the velocity of the transverse wave traveling along the string itself. This speed is determined by the mechanical properties of the string:
* **Tension ($T$):** How tightly the string is stretched.
* **Linear Mass Density ($\mu$):** The mass per unit length of the string.

If the tension were increased, the wave speed would increase, subsequently raising the fundamental frequency (pitch) of the note.

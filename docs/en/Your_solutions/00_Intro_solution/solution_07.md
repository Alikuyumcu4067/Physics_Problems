## 7. Basic Calculus - Derivatives

**Problem:**
Find the derivative of the position function $x(t) = 5t^2 + 3t + 10$ with respect to time $t$. (This gives the velocity $v(t)$).

**Key Formula:**
The Power Rule: $\frac{d}{dt}(at^n) = n \cdot at^{n-1}$
The derivative of a constant is $0$.

**Step-by-Step Solution:**

1. **Given Function:**
   $x(t) = 5t^2 + 3t + 10$

2. **Differentiate each term:**
   * $\frac{d}{dt}(5t^2) = 2 \cdot 5t^{(2-1)} = 10t$
   * $\frac{d}{dt}(3t) = 1 \cdot 3t^{(1-1)} = 3 \cdot t^0 = 3$
   * $\frac{d}{dt}(10) = 0$

3. **Combine the results:**
   $v(t) = \frac{dx}{dt} = 10t + 3$

**Final Result:**
The derivative of the function is:
$$\mathbf{v(t) = 10t + 3}$$

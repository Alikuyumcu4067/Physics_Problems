## 8. Basic Calculus - Integrals

**Problem:**
Find the integral of the velocity function $v(t) = 10t + 3$ with respect to time $t$. (This gives the position function $x(t)$).

**Key Formula:**
The Power Rule for Integration: $\int at^n \, dt = \frac{at^{n+1}}{n+1} + C$
The integral of a constant $a$ is $at + C$.

**Step-by-Step Solution:**

1. **Given Function:**
   $v(t) = 10t + 3$

2. **Integrate each term:**
   * $\int 10t^1 \, dt = \frac{10t^{(1+1)}}{2} = 5t^2$
   * $\int 3 \, dt = 3t$

3. **Combine the results:**
   Don't forget the constant of integration ($C$):
   $x(t) = \int (10t + 3) \, dt = 5t^2 + 3t + C$

**Final Result:**
The position function (the integral) is:
$$\mathbf{x(t) = 5t^2 + 3t + C}$$

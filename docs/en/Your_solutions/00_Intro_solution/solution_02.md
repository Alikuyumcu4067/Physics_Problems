## 2. Systems of Equations

**Problem:**
Solve the following system of equations for $x$ and $y$:
1) $2x + 3y = 12$
2) $x - y = 1$

**Solution:**
We can use the substitution method. From equation (2), let's express $x$ in terms of $y$:
$x = y + 1$

Now, substitute this value of $x$ into equation (1):
$2(y + 1) + 3y = 12$
$2y + 2 + 3y = 12$

Combine the $y$ terms:
$5y + 2 = 12$
$5y = 10$
$\mathbf{y = 2}$

Now that we have $y$, find $x$ by substituting it back into our $x$ equation:
$x = 2 + 1$
$\mathbf{x = 3}$

**Final Answer:**
The solution to the system is $(x, y) = (3, 2)$.
